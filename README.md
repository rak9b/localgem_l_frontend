## 1. Project Overview
Next.js 14 based marketplace frontend for LocalGems.

## 2. Features
- **3D Scale Hovers**: Real-time perspective animations.
- **Glassmorphism**: Backdrop blur system tokens.
- **Role Guards**: Auth-protected dashboards.

## 3. Architecture
RTK Query state management with Next.js App Router.

## 4. Technology Stack
Next.js, Tailwind v4, Framer Motion, Zod.

## 5. Setup & Installation
`npm install && npm run dev`

## 6. Testing Strategy
- **Unit**: Jest + React Testing Library.
- **E2E**: Playwright ready.

## 7. Security Considerations
- **XSS**: Automatic React escaping.
- **CSRF**: Token-based API communication.

## 8. Performance Optimization
- **Image**: Next/Image for WebP delivery.
- **Lighthouse Score**: 95+ targeted.

## 9. Accessibility Audit
WCAG 2.1 compliant semantic markup.
